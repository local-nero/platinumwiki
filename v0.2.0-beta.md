# PlatinumDL v0.2.0-beta

## 🌟 New

* Added a complete submission review workflow for staff.
* Introduced a modern review modal with all submission information in one place.
* Added submission history tracking, allowing staff and players to view every status change.
* Added notes to staff reviews.
* Added device selection to submissions.
* Added mod menu selection to submissions (None, PolzHax, Other).
* Added optional Custom ID field.
* Added optional Raw Footage link.
* Added optional Notes field.
* Added editable submissions for players while a submission is still pending.
* Added submission locking to prevent multiple staff members reviewing the same record simultaneously.
* Added player profile improvements.
* Added searchable country picker with support for all ISO countries.
* Added support for England, Scotland, and Wales country selections.
* Added automatic country flag support across the website.
* Added Discord ID tracking for submitted records.
* Added groundwork for the upcoming Stats Viewer.

## 📊 Stats Viewer

* Introduced the new Stats Viewer framework.
* Player pages now support Discord-linked profiles.
* Added completed records tracking.
* Added placed levels tracking.
* Added first victor tracking.
* Added points calculation.
* Added framework for Pack completion support.
